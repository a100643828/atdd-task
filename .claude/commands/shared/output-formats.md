# 輸出格式參考

## 通用框架

```
┌──────────────────────────────────────────────────────┐
│ {emoji} {標題}                                       │
├──────────────────────────────────────────────────────┤
│ {內容區塊}                                           │
└──────────────────────────────────────────────────────┘
```

## 任務建立

**必要欄位**：類型、專案、標題、ID
**可選欄位**：路徑（test 類型）、Epic 資訊、Jira 連結

## 階段轉移

**必要欄位**：任務識別、從/到階段、下一個 Agent
**可選欄位**：/clear 提示、Jira 連結

## 結案完成

**必要欄位**：任務識別、Commit hash、狀態、Metrics
**可選欄位**：Epic 同步結果、知識更新提案、Jira 連結

## 錯誤/警告

**必要欄位**：問題描述、可用選項
**可選欄位**：詳細原因

## 分析結果

**必要欄位**：Domain、信心度
**可選欄位**：範圍、場景清單、待澄清問題、Jira 連結
